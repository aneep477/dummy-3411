Sistem Pemarkahan MPU3411 Pro - IPG Kampus Darulaman
Sistem Pemarkahan MPU3411 adalah aplikasi berasaskan web yang dibangunkan untuk memudahkan para pensyarah di IPG Kampus Darulaman merekod, mengurus, dan memantau prestasi pelajar bagi kursus MPU3411 Kokurikulum Olahraga.

Aplikasi ini menggantikan penggunaan borang Excel manual dengan antaramuka sistem yang lebih moden, responsif, dan mempunyai ciri validasi markah serta rujukan rubrik digital secara terus.

Dashboard & Statistik

Paparan ringkas jumlah pelajar dan status pengisian markah (Ujian, HP3, HP7).

Jadual ringkasan markah keseluruhan pelajar mengikut kelas.

Statistik visual (kad warna) untuk pemantauan pantas.

Modul Pengisian Markah

Ujian (30%): Borang input untuk markah teori.

Amali HP3 (50%): Borang input kemahiran Balapan (Lari Pecut) dan Padang (Lontar Peluru).

Penyediaan HP7 (20%): Borang input untuk penyediaan trek dan sektor.

Rubrik Digital Bersepadu (5 Peringkat)

Rujukan rubrik boleh diakses terus melalui butang "LIHAT RUBRIK".

Memaparkan kriteria pemarkahan 5 Peringkat (Amat Cemerlang hingga Lemah).

Tab berasingan untuk Acara Balapan dan Acara Padang bagi memudahkan rujukan pensyarah.

Ciri Keselamatan & Validasi

Sistem Login: Akses dikawal mengikut peranan (Admin & Pensyarah) dan akses kelas tertentu.

Validasi Input: Kotak markah akan bertukar merah dan memberi amaran jika markah dimasukkan melebihi had maksimum.

Penyimpanan Awan: Data disimpan terus ke Google Sheets secara real-time.
